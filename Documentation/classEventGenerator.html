Detailed Description

Generates number of Event and saves them to ROOT type or text files.

The THERMINATOR 2 main settings file therminator.ini options used here:

Keyword Description
NumberOfEvents number of events to be generated
IntegrateSamples number of Monte-Carlo samples used by Integrator
EventDir general director to store events, subdirectory is added by Model
EventFileType type of files to save events
"root"ROOT file : eventXXX.root [default]
"text" TEXT file : event.txt
"root&text" both ROOT and TEXT files
LogFile name and location of the LOG file

EventGenerator creates an Event object. For a given number of events the Event is called to generate primordial particles and to decay resonances. The Particle list is than saved to file, either a ROOT type file or a TEXT file or both.

In case the event directory already has some eventXXX.root EventGenerator save new files with the name counter set to continue the number sequence.

EventGenerator also create a log entry to a CALM log file [default: calm.log]. The log entry consists of:

  • current time and date,
  • Model number (sModel)
  • directory the events are stored
  • initial event file number in this program run
  • last event file number in this program run

Creates a temporary file in the main THERMINATOR 2 directory called event_<aPID>.tmp.

Default name is event_0.tmp.The file contains two lines:

  • current directory where the event files are stored,
  • number of event files located at that directory, including previously generate files.
