Detailed Description

Class containing particle of ParticleType space-time and energy-momentum coordinates.

Class has information about the PDG Monte-Carlo number of the ParticleType, father particle and the original primordial particle To back trace the whole decay cascade Particle stores the unique particle number of the particle in event and its fathers. Particle also holds information if the particle has decayed.
